# Faglig Fornyelse – Next.js Projekt

> **Skole/uddannelsesprojekt** der demonstrerer opbygning af en moderne webapplikation med [Next.js](https://nextjs.org/).

---

## 📋 Opgavebeskrivelse

Formålet med dette projekt er at sætte sig ind i og anvende **Next.js** som framework til at bygge en moderne, serverrenderet React-applikation. Projektet er en del af den faglige fornyelse og skal vise forståelse for:

- **App Router** – filbaseret routing med Next.js 13+
- **Server Components** – rendering på serveren for bedre ydeevne
- **Client Components** – interaktive komponenter i browseren
- **Datahentning** – brug af `fetch`, API-ruter og server actions
- **Styling** – CSS Modules, Tailwind CSS eller anden stylingmetode
- **Deployment** – publicering af applikationen (fx via Vercel)

---

## 🎯 Læringsmål

Igennem projektet ønskes der opnået erfaring med:

1. Opsætning af et Next.js-projekt fra bunden
2. Forståelse af forskellen mellem server- og klientkomponenter
3. Implementering af sider og layouts med App Router
4. Hentning og visning af data fra et eksternt API eller en database
5. God kodestruktur og komponentbaseret tænkning

## 🚀 Kom i gang

### Forudsætninger

- [Node.js](https://nodejs.org/) version 18 eller nyere
- [npm](https://www.npmjs.com/), [yarn](https://yarnpkg.com/) eller [pnpm](https://pnpm.io/)

### Installation

1. Klon dette repository:
   ```bash
   git clone https://github.com/JackSHansen/Faglig-fornyelse-Next.js.git
   cd Faglig-fornyelse-Next.js
   ```

2. Installér afhængigheder:
   ```bash
   npm install
   # eller
   yarn install
   ```

3. Start udviklingsserveren:
   ```bash
   npm run dev
   # eller
   yarn dev
   ```

4. Åbn [http://localhost:3000](http://localhost:3000) i din browser.

# Opgavebeskrivelse – Personlig Dashboard Web App i Next.js

## Formål

Formålet med dette projekt er at udvikle en moderne webapplikation ved brug af Next.js. Projektet skal give en forståelse for, hvordan Next.js adskiller sig fra almindelig React, herunder arbejde med routing, server-side funktionalitet og datahåndtering.

Derudover er målet at opbygge en interaktiv og brugervenlig applikation, der samler flere funktioner i ét dashboard.

---

## Projektbeskrivelse

Jeg vil udvikle en personlig dashboard web app, hvor brugeren kan få et hurtigt overblik over relevante informationer og værktøjer i hverdagen.

Applikationen vil bestå af forskellige komponenter (widgets), som brugeren kan interagere med.

---

## Funktionelle krav

Applikationen skal som minimum indeholde følgende funktioner:

### 1. Dashboard (forside)
- En oversigtsside med widgets
- Visning af dato og tid
- En simpel og brugervenlig UI

### 2. Notes funktion
- Brugeren kan oprette noter
- Mulighed for at slette noter
- Noter gemmes lokalt (fx localStorage eller via API)

### 3. To-do liste
- Oprette opgaver
- Markere opgaver som færdige
- Slette opgaver

### 4. Vejr-widget
- Hente data fra en ekstern API
- Vise temperatur og vejrforhold for en valgt by

---

## Ekstra funktioner

### 🌙 Dark Mode
- Brugeren kan skifte mellem light og dark theme
- UI tilpasser sig automatisk
- Indstillingen gemmes

### 🖱️ Drag and Drop
- Brugeren kan flytte rundt på widgets på dashboardet
- Giver en mere personlig og dynamisk brugeroplevelse

---

## Tekniske krav

Projektet skal udvikles med følgende teknologier:

| Teknologi | Beskrivelse |
|---|---|
| **Next.js** | Rammeværk til React |
| **React** | UI-bibliotek |
| **SCSS** | Styling |
| **JavaScript / TypeScript** | Programmeringssprog |

Derudover skal følgende Next.js-features anvendes:

- Routing (App Router eller Pages Router)
- Komponentopbygning
- Data fetching
- API routes (hvis relevant)

---

## Læringsmål

Med dette projekt ønsker jeg at opnå:

- ✅ Forståelse for Next.js struktur og opbygning
- ✅ Erfaring med forskellen mellem client-side og server-side rendering
- ✅ Arbejde med API'er og datahåndtering
- ✅ Opbygning af interaktive brugerinterfaces
- ✅ Implementering af ekstra funktionalitet som dark mode og drag-and-drop
